Sum Of All Fears

Goal: An app to check if two given numbers add up to a target number

Purpose: this exercise builds and array and then uses an external value to test the elements of that array against a certain condition


One of the most important things to be able to do when working with arrays is to somehow access and utilize their content. This can be as simple as something like this:

      var cars = ["Saab", "Volvo", "BMW"];
      document.getElementById("demo").innerHTML = cars;
This bit of code would target some element in the document with the id of "demo" and display the contents of the array "cars". Which is handy enough, but utilizing many different mechanisms, we can do so much more as shown in this simple challenge. At a high level, what we want to achieve here being able to collect a set of numbers from the user, build them into an array, and then check if any two of those numbers add up to an another value outside of the array also supplied by the user. A more detailed description will be available in the "Code" section.
