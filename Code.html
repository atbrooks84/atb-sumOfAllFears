Sum Of All Fears

Goal: An app to check if two given numbers add up to a target number


    document.getElementById("soafBtn").addEventListener('click', function () {

    let soafArray = [];
    let k = parseInt(document.getElementById("kvalue").value);

    soafArray.push(parseInt(document.getElementById("soafNum1").value), parseInt(document.getElementById("soafNum2").value), parseInt(document.getElementById("soafNum3").value), parseInt(document.getElementById("soafNum4").value), parseInt(document.getElementById("soafNum5").value), parseInt(document.getElementById("soafNum6").value));

    for (index = 0, found = false; index < soafArray.length - 1; index++) {
    let position = soafArray.indexOf(k - soafArray[index], index + 1);
    if (position > 0) {
    found = true;
    if (found = true) {

    document.getElementById("soafOut").innerHTML = (`${k} was found by adding ${soafArray[index]} and ${soafArray[position]}`);
    break;

    }
    } else {

    document.getElementById("soafOut").innerHTML = (`${k} was not found`);
    }



    }


    })
What this code is achieving is first creating an empty array and then populating it using the six numbers supplied by the user. The seventh value, the "k" value, is stored in a variable outside of the array. Using a for loop to increment over the array, when then use a new variable "position" to check if the current selected element of the array can be equal to the "k" value if added to any other element. We do this by indexing over each element a second time within that comparison. The if/else statement then has to conditions. If "position" plus "index" equals "k", then the loop breaks and the two successful values are displayed on the document. Otherwise, a simple failure message is displayed.
